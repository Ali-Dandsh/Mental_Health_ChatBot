How to run the bot

pip install -r requirements.txt
uvicorn main:app --reload

-------------------------------------------
Chatbot API Documentation

1. Start server:
   uvicorn main:app --reload

2. Endpoint:
   POST /chat

Request:
{
"message": "your message"
}

Response:
{
"reply": "AI response",
"emotion": "detected emotion",
"practice": ["exercise1", "exercise2"]
}

3. Get chat history:
   GET /history
